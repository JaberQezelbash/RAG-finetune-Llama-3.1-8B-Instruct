# RAG-based Llama Finetuning
This project builds a RAG-based multilingual IT support chatbot using Meta Llama-3.1-8B-Instruct as the generator, BGE-M3 for retrieval, BGE reranking, and FAISS vector search. Fine-tunes the generator with QLoRA SFT on the support-ticket data and evaluated performance using retrieval quality, reranks evidence relevance, lexical F1, ROUGE-L, and generates response quality metrics.
